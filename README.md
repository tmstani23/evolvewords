# evolvewords
Genetic Algorithm:
Evolve a word phrase in your browser.

To run:
1. Click the clone or download dropdown
2. Download as zip
3. Extract onto your desktop
4. Run the index.html file
5. Refresh your browser a few times if it doesnt begin running immediately.
